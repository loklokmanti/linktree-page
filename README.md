# Aesthetic Link Page

This is a modern, responsive "Linktree-style" page for your social media bio.

## How to use

1. Open `index.html` in any web browser to view your page.
2. To customize:
   - **Profile Image**: Replace `profile_placeholder.png` with your own image (keep the same name or update the `src` in `index.html`).
   - **Links**: Open `index.html` in a text editor. Look for the `<section class="links-section">`. You can copy/paste the `<a href="..." class="link-card">` blocks to add more links. Change the `href="#"` to your actual URL.
   - **Colors**: Open `style.css` and modify the variables at the top (`:root`) to change the color scheme.
   
## Features

- Glassmorphism design
- 3D Tilt effect on hover
- Responsive layout (works on mobile and desktop)
- Fast performance (no heavy frameworks)
